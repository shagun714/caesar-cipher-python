# Caesar Cipher 🔐

A Python project that encrypts and decrypts messages using the Caesar Cipher algorithm.

## What it does
This program shifts letters in a message by a chosen number of positions in the alphabet to encrypt or decrypt text.

Example:
hello → khoor  (shift = 3)

## How to run it
1. Install Python
2. Download the project
3. Run:

python Untitled-1.py

## Example output

Enter text: hello
Enter shift: 3

Encrypted text: khoor
